Telegram Bot Setup

Get photo alerts and remote control your security camera via Telegram in 5 minutes.

What You Need

  • Telegram app installed on your personal phone, tablet, or computer (also works via web.telegram.org in any browser)
  • A Telegram account
  • 5 minutes

Step 1: Create Your Telegram Bot

1 Open Telegram on your personal phone

2 Search for @BotFather and tap Start

BotFather is Telegram's official bot for creating other bots. Look for the blue verified checkmark.

3 Send the command /newbot

4 Enter a display name (e.g., "My Security Camera")

5 Enter a username ending in bot (e.g., "mysecuritycam_bot")

The username must be unique. If taken, try adding numbers.

BotFather will reply with your Bot Token — a long string like:

1234567890:ABCdefGHIjklMNOpqrSTUvwxYZ123456789
Save this token! You'll enter it in the app's Settings. Don't share it with anyone — it controls your bot.

Step 2: Get Your Chat ID

1 Open your new bot in Telegram (search for the username you just created)

2 Tap Start and send any message (e.g., "hello")

3 Open this URL in your browser (replace YOUR_TOKEN with your actual token):

https://api.telegram.org/botYOUR_TOKEN/getUpdates

4 Find "id" inside the "chat" object:

"chat":{"id":987654321,"first_name":"Your Name"}

The number (e.g., 987654321) is your Chat ID.

Can't find the Chat ID? Make sure you sent a message to the bot FIRST, then refresh the getUpdates URL.

Step 3: Enter in the App

1 Open HumanDetector AI → tap Settings

2 Paste your Bot Token and Chat ID

Fields are masked by default (shown as dots). Tap the 👁 eye button to reveal them while pasting.

3 Tap Test Connection

You should see "✅ Connection OK" and receive a test message in your Telegram chat.

4 Tap Save Settings

Optional: Group Chat for Family Monitoring

Want family members or a security team to also see alerts and send commands?

1 Create a Telegram group (e.g., "Home Security")

2 Add your bot to the group (search for its username)

3 Send a message in the group

4 Visit the getUpdates URL again — look for the group chat ID (it's a negative number like -1001234567890)

5 Use this group Chat ID in the app instead of your personal Chat ID

All group members can now see alerts and send commands. Use device names (e.g., /unlock_kitchen) to control specific phones.

Multi-Phone Setup

You can use the same bot and chat ID on multiple phones. Give each phone a unique Device Name in Settings (e.g., "kitchen", "frontdoor", "garage").

Send /help to see clickable commands for each phone. Tap any command to send it instantly.

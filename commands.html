Remote Commands

Control your security cameras from anywhere via Telegram.

Send /help to your bot to get a clickable list of all commands customized with your device name. Tap any command to send it instantly.

Commands use underscore format for one-tap sending: /unlock_kitchen. You can also type with spaces: /unlock kitchen. Both work.

Broadcast Commands

These work without a device name — all connected phones respond.

CommandWhat it does
/photoTake and send a photo with detection boxes. Shortcut: /p
/statusFull status: detection state, engine, battery, all settings. Shortcut: /s
/helpShow clickable command list for all connected phones. Shortcut: /h
/alarmSound the alarm on all phones. Shortcut: /a

Device-Specific Commands

Replace NAME with your device name (e.g., kitchen, frontdoor).

🔒 Security

⚙ Settings

Pro Tips

  • Pin the /help message in your Telegram chat for instant access to all commands
  • During theft mode, every photo has a clickable /unlock — tap any photo to stop it
  • Use /status to check battery and settings on all phones at once
  • Heartbeat pings confirm the phone is still running — set to 15 min for peace of mind
  • Wrong device? Commands with another device's name are silently ignored — no accidental triggers
